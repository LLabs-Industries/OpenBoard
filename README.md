# OpenBoard

OpenBoard is a personal Arduino repository for storing code, circuits, and hardware experiments.  
This repo serves as a growing archive of sketches, prototypes, and rapid hardware concepts.

Everything here is shared openly and unlicensed — use it, modify it, break it, improve it.

## What’s Inside
- Arduino sketches and firmware
- Circuit designs and wiring notes
- Rapid prototypes and hardware experiments
- Miscellaneous embedded projects
